# FujiFish Chess
This is a WIP chess server for Fujinet clients. It uses [StockFish](https://stockfishchess.org/)
as a Chess engine for single player games. I"m also using [python-chess](https://python-chess.readthedocs.io/en/latest/) 
to manage the board.


Supported Platforms
* Atari 8-bit Computers
* MS-DOS
